# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar
```
npm install
```
3. Tener Nest CLI instalado
```
npm i -g @nestjs/cli
```
4. Levantar la base de datos
```
docker-compose up -d
```

# Otros
Descargar imagen de Mongo 5.0.0 y postgres 14
```
docker pull mongo:5.0.0
docker pull postgres:14.3
```
